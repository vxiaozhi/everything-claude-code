# 学习命令

从当前会话中提取可复用的模式。

## 模式提取

1. **识别模式**
   - 错误解决方案
   - 用户更正
   - 框架变通方法
   - 调试技术
   - 项目特定约定

2. **保存到技能**
   - 存储在 `~/.claude/skills/learned/`
   - 添加元数据（日期、上下文）
   - 创建可重用的片段

3. **将来检索**
   - 在未来会话中自动加载
   - 搜索模式

## 使用

```bash
/learn
```

命令将：
- 分析会话历史
- 识别提取候选
- 要求确认
- 保存到可学习技能

## 模式类型

| 类型 | 描述 |
|---------|-------------|
| `error_resolution` | 如何解决特定错误 |
| `user_corrections` | 从用户更正中学到的模式 |
| `workarounds` | 框架/库变通方案 |
| `debugging_techniques` | 有效的调试方法 |
| `project_specific` | 项目特定约定 |

与 **continuous-learning** 技能一起使用。
