# Eval 命令

运行评估以验证代码质量并检测回归。

## 评估类型

### 功能评估
测试新能力：
- Claude 能做它以前不能做的事吗？
- 成功标准：通过/失败

### 回归评估
确保稳定性：
- 现有功能是否中断？
- 成功标准：100% 通过

## 工作流程

```bash
/eval define feature-name    # 定义评估
/eval check feature-name     # 运行评估
/eval report feature-name    # 生成报告
```

## 指标

- **pass@1**：第一次尝试的成功率
- **pass@3**：3 次尝试内的成功率
- **pass^3**：3 次连续成功（可靠性）

## 评估驱动开发

1. **定义** - 首先编写评估
2. **实施** - 编写代码以通过评估
3. **评估** - 持续运行评估
4. **改进** - 迭代直到 pass@3 > 90%

与 **eval-harness** 技能一起使用。
