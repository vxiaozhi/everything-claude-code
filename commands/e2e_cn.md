# E2E 测试命令

使用 Playwright 生成和维护端到端测试。

## 工作流程

1. **确定旅程**
   - 识别关键用户流程
   - 将旅程映射到步骤

2. **生成测试**
   - 使用 `e2e-runner` 代理
   - 创建独立、可重用的测试
   - 添加适当的断言

3. **运行和验证**
   ```bash
   npx playwright test
   ```

4. **管理工件**
   - 捕获屏幕截图
   - 记录视频
   - 保存跟踪以进行调试

## 使用

```bash
/e2e
```

代理将：
- 分析代码库中的用户流程
- 生成 Playwright 测试
- 设置适当的页面对象模型
- 添加不稳定测试处理
- 配置工件捕获
