# 构建修复命令

修复构建错误和 TypeScript 类型问题。

## 工作流程

1. **诊断**
   - 运行构建命令
   - 捕获所有错误
   - 按类型分类错误

2. **修复**（最小差异）
   - 添加缺失的类型
   - 修复导入
   - 添加空值检查
   - 使用 `build-error-resolver` 代理

3. **验证**
   - 重新运行构建
   - 确认通过

## 使用

```bash
/build-fix
```

代理将自动：
- 运行 `npm run build` 或 `pnpm build`
- 捕获所有 TypeScript 错误
- 使用最小更改进行修复
- 验证构建成功
